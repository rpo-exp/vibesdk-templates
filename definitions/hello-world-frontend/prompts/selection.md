# Template Selection

Environment-aware frontend template based on a simple static hello-world app.
Defaults include visible rPotential branding and rPotential-style domain conventions.

Use when:
- You want a clean status/overview page with deployment context
- You need branch/domain display logic in the UI
- You want a lightweight starting point for public-facing pages

Avoid when:
- You need Durable Objects, KV, or advanced backend orchestration
- You need a large component-heavy dashboard out of the box

Built with:
- Vite + React + TypeScript
- Cloudflare Workers static asset serving pattern
- GitHub Actions workflows for Azure Static Web Apps deployment
